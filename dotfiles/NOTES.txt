 2.10 | BLUETOOTH ON DUAL BOOT
 - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
 * Pair bluetooth device on Arch Linux
 * Reboot into Windows and pair the device on Windows
 * Download PsTools (https://download.sysinternals.com/files/PSTools.zip)
 * Extract PsExec64.exe
 * Run cmd as admin in the folder where PsExec64.exe is located
 * Run the command ".\PsExec64.exe -s -i regedit.exe"
 * In the registry editor, navigate to "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\BTHPORT\Parameters\Keys"
 * Copy the key of the bluetooth device (in the "data" field)
 * Reboot into Arch Linux
 * Edit the file /var/lib/bluetooth/<BT-Adapter-MAC-Address>/info and paste the key copied from windows into the Key field (without spaces between the characters)

 2.11 | SERVER SETUP
 - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
 * Static IP:
    * Get list of network devices:
    $ nmcli device status
    * List current connection info:
    $ nmcli device show <device id>
    * Take note of IP4.ADDRESS, IP4.GATEWAY, IP4.DNS[1], IP4.DNS[2]
    $ nmtui
    * Edit your connection, set IPv4 CONFIGURATION to Manual and fill in desired static IP + gateway and DNS from earlier
 * SSH (default port 22):
    $ sudo pacman -S openssh
    $ systemctl enable sshd.service
    $ systemctl start sshd.service
    * From the client computer:
    $ (client) ssh-keygen -t rsa
    $ (client) ssh-copy-id <username>@<server ip>
    * Back to the server computer:
    $ sudo vim /etc/ssh/sshd_config
    * Disable root login, set desired port and disable password login (uncomment and change lines):
    * Port <desired port>
    * PermitRootLogin no
    * PermitEmptyPasswords no
    * PasswordAuthentication no
    * AuthenticationMethods publickey
    $ sudo pacman -S fail2ban
    $ systemctl enable fail2ban.service
    $ systemctl start fail2ban.service
    * (on client) ssh <username>@<server ip> -p <port>
 * nginx (port 80/443):
    $ chmod 701 /home/<username>
    $ sudo pacman -S nginx
    $ sudo vim /etc/nginx/nginx.conf
    * See "example/nginx.conf" for an example file with two domains (replace <domain1> and <domain2> with domain names)
    $ sudo pacman -S certbot certbot-nginx
    $ sudo certbot --nginx
    $ systemctl enable nginx.service
    $ systemctl start nginx.service
    $ systemctl enable certbot-renew.timer
 * PostgreSQL (default port 5432):
    $ sudo pacman -S postgresql
    $ sudo -u postgres vim /var/lib/postgres/data/postgresql.conf
    * listen_addresses = '*'
    $ sudo -u postgres vim /var/lib/postgres/data/pg_hba.conf
    * host    all             all             0.0.0.0/0            md5
    $ sudo -iu postgres
    $ initdb --locale nb_NO.UTF-8 -E UTF8 -D /var/lib/postgres/data
    $ systemctl enable postgresql
    $ systemctl start postgresql
    $ psql
    $ CREATE USER <username> WITH PASSWORD '<password>';
    $ CREATE DATABASE <db name> OWNER <username>;
    $Â GRANT ALL PRIVILEGES ON DATABASE <db name> TO <username>;
    * (on client) psql -h <server ip>:5432 -U <username> -d <db name>
    * (optional) Add SSL support:
        $ sudo -u postgres vim /var/lib/postgres/data/postgresql.conf
        * Use SSL cert from nginx certbot config:
            ssl = on  
            ssl_cert_file = '<fullchain.pem from certbot (aquired during nginx setup)>'
            ssl_key_file = '<privkey.pem from certbot (aquired during nginx setup)>'
            ssl_prefer_server_ciphers = on
            ssl_dh_params_file = '/etc/letsencrypt/ssl-dhparams.pem'
 * Firewall (ufw):
    $ sudo pacman -S ufw
    $ sudo ufw default deny incoming
    $ sudo ufw default allow outgoing
    $ sudo ufw allow ssh
    $ sudo ufw allow http
    $ sudo ufw allow https
    $ sudo ufw allow 5432/tcp
    $ systemctl enable ufw.service
    $ systemctl start ufw.service
 * Logging (logrotate / lnav / goaccess):
    $ sudo pacman -S lnav goaccess logrotate
    $ sudo vim /etc/logrotate.d/nginx
    * See "example/nginx" for an example file
    $ systemctl enable logrotate.timer
    $ systemctl start logrotate.timer
    * Example commands for viewing log files:
    $ lnav /var/log/nginx/error.log
